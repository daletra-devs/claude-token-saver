# claude-token-saver

A custom Claude Desktop skill designed to maximize token savings (input and output) by enforcing code minification, unified diff outputs, and a strict telegraphic response style.

## How to install & use

1. Download the custom skill file from this repository.
2. Open your Claude Desktop application and go to the **Custom Skills** or **Extensions** menu.
3. Click on **Upload a skill** and simply drag and drop the downloaded file.
